# Phase 2: Minimax AI Module

## Purpose
Add an AI opponent using Minimax with Alpha-Beta pruning.

## Files
- `minimax.py` - Minimax algorithm, heuristic scoring.
- `ai_player.py` - Interactive human vs AI runner.
- `test_ai.py` - Simulate AI vs random players to sanity-check.

## How to run
From the project root `connect4_ai/`:

Play vs AI (interactive):
```bash
python minimax_ai/ai_player.py
